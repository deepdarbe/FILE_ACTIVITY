Standart Referansi

Zorunlu Kurallar (Requirements):
R1: Dosya adinda bosluk olmamali
R2: Ilk karakter ASCII harf (a-z, A-Z) olmali
R3: Sadece ASCII harf, rakam, tire, alt cizgi, nokta kullanilmali
R4: Tek nokta + uygun uzanti (3+ harf: jpg, pdf, tif) ile bitmeli
En Iyi Uygulamalar (Best Practices):
B1: Dosya adi 31 karakter veya daha kisa olmali
B2: Toplam yol 256 karakteri gecmemeli
B3: Base'de nokta kullanilmamali
B4: Tum harfler kucuk olmali
B5: Bolumler alt cizgi (_) ile ayrilmali
B6: Dizin adlarinda nokta olmamali

⚠️ Geri Yukleme — DIKKAT

Bu islem mevcut veritabanini secilen snapshot ile tamamen degistirir. Onceee dashboard servisi durdurulmali; aksi takdirde sunucu istek reddeder. Devam etmeden onceee:
  • Tum acik dashboard sekmelerini kapatin.
  • Servisi durdurun (net stop file_activity).
  • Bu islem GERI ALINAMAZ.

📡 Syslog Konfigurasyonu

Salt-okunur — degisiklikler icin config.yaml dosyasini duzenleyin.
integrations:
  syslog:
    enabled: false           # true yapinca forwarder calisir
    host: "siem.firma.local" # Hedef SIEM hostu
    port: 514                # 514 (UDP), 6514 (TLS), 1468 (Splunk TCP)
    transport: "udp"         # udp | tcp | tcp_tls
    format: "rfc5424"        # rfc5424 | cef
    facility: "local0"
    queue_max: 10000
    tls_ca_path: ""          # Sadece transport=tcp_tls icin

🛡️ GDPR Subject Export (Article 17 / 30)

Kisisel veri sahibinin (e-posta, TC, ad-soyad vb.) gectigi tum dosyalari listele. CSV cikti olarak indirilir; yetkili paydaslar disinda paylasilmamalidir.
Tam veya kismi metin — snippet aramasi yapilir.

📋 Yeni Retention Policy
