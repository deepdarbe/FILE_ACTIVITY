Dosya Tarama ve Analiz

Windows dosya paylasimlarini (UNC yollari) tarar, dosya erisim zamanlarini, boyutlarini ve turlerini analiz eder.

  • Win32 API ile dosya ozelliklerini okuma
  • Artimsal (incremental) tarama destegi
  • Coklu dosya sunucu kaynak destegi
  • Gizli ve sistem dosyalarini filtreleme
📈

Detayli Raporlama

Dosya erisim sikligi, dosya turu dagilimi ve boyut dagilimi raporlari olusturur.

  • 30/60/90/180/365 gun erisim sikligi
  • Dosya turu bazinda analiz
  • Boyut kategorisine gore dagilim
  • HTML, JSON ve CSV cikti formatlari
📦

Akilli Arsivleme

Belirlenen politikalara gore dosyalari arsiv hedefine tasir, dizin yapisini tamamen korur.

  • Kopyala-Dogrula-Sil guvenligi
  • SHA-256 checksum dogrulamasi
  • Dizin yapisi tam koruma
  • Manuel ve zamanlanmis arsivleme
🔎

Arsiv Indeksi ve Geri Yukleme

Arsivlenen dosyalar icin aranabilir indeks olusturur, ihtiyac halinde hizlica geri yukleme yapar.

  • SQLite FTS5 tam metin arama
  • Dosya adi, yol, uzanti ile arama
  • Tek tikla geri yukleme
  • Arsiv istatistikleri
⏰

Zamanlama ve Otomasyon

Tarama ve arsivleme islemlerini otomatik olarak zamanlanmis gorevlerle calistirir.

  • Dahili APScheduler zamanlayici
  • Windows Task Scheduler entegrasyonu
  • Cron ifadesi destegi
  • Windows Service olarak calistirma
🌐

Web Dashboard

FastAPI tabanli web arayuzu ile tum islemleri goruntuleyebilir ve yonetebilirsiniz.

  • Canli durum raporu
  • Interaktif grafikler ve tablolar
  • Arsiv arama ve geri yukleme
  • Kaynak ve politika yonetimi

🧩 Modul Detaylari

🔍

scanner/file_scanner.py

Dosya paylasimlarini recursive olarak tarar.

  • scan_source(source) - Ana tarama fonksiyonu
  • recursive_scandir(path) - os.scandir jeneratoru
  • should_skip(entry) - Filtre kontrolu
  • Batch INSERT (500 kayit/islem)
  • Artimsal tarama destegi
  • PermissionError toleransli
⚙

scanner/win_attributes.py

Windows dosya ozelliklerini pywin32 ile okur.

  • get_file_times(path) - FileTimesInfo dondurur
  • GetFileAttributesEx API kullanimi
  • creation_time, access_time, modify_time
  • Dosya boyutu (high+low DWORD)
  • os.stat() fallback destegi
  • NtfsDisableLastAccessUpdate kontrolu
📊

analyzer/frequency_analyzer.py

Dosya erisim sikligini gun araligina gore analiz eder.

  • analyze_frequency(source_id, buckets)
  • Yapilandirılabilir gun araliklari: 30/60/90/180/365
  • SQL GROUP BY tabanli verimli sorgular
  • Son erisim vs. son degisiklik zamani
📄

analyzer/type_analyzer.py

Dosya turlerine gore dagilim analizi.

  • analyze_types(source_id)
  • Uzanti bazinda: sayi, toplam boyut, ortalama
  • En buyuk, en kucuk, en eski, en yeni
  • Grafik verisi olarak cikti
📏

analyzer/size_analyzer.py

Dosya boyutu dagilimi analizi.

  • analyze_sizes(source_id)
  • tiny (<100KB), small (<1MB), medium (<100MB)
  • large (<1GB), huge (>1GB)
  • Kumulatif boyut ve dosya sayisi
📦

archiver/archive_engine.py

Guvenli arsivleme motoru.

  • archive_file(source, file, dest)
  • Kopyala → Dogrula → Kaydet → Sil
  • SHA-256 butunluk kontrolu
  • Dizin yapisi tam koruma
  • Bos dizin temizligi
  • Dry-run onizleme modu
📜

archiver/archive_policy.py

Arsiv politika kural motoru.

  • evaluate(file, policy) - Kural degerlendirme
  • last_access_older_than_days
  • last_modified_older_than_days
  • file_size_greater_than / less_than
  • extension_in / extension_not_in
  • path_pattern (glob eslesme)
  • AND/OR kural birlestirme
♻

archiver/restore_engine.py

Arsivden geri yukleme motoru.

  • restore_by_id(archive_id)
  • restore_by_path(original_path)
  • Orijinal dizin yapisini yeniden olusturur
  • Arsiv indeksini gunceller

⚠ Onemli Notlar

⚠

NtfsDisableLastAccessUpdate

Windows Vista'dan beri varsayilan olarak last access time guncellemesi kapalidir. Tarayici bu registry anahtarini kontrol eder ve uyari verir. Eger kapali ise, last_modify_time alternatif kriter olarak kullanilmalidir.

HKLM\SYSTEM\CurrentControlSet\Control\FileSystem\NtfsDisableLastAccessUpdate

🔒

Guvenlik

Arsivleme islemi Kopyala-Dogrula-Sil deseni kullanir. Kaynak dosya ancak ve ancak hedef dosyanin SHA-256 checksum'i eslestiginde silinir. Bu, veri kaybi riskini minimize eder.

📊

Performans

os.scandir() kullanimi ag paylasimlari taranirken stat() cagrilarini azaltir. Batch INSERT (500 kayit) SQLite performansini optimize eder. WAL modu okuma/yazma esitligini saglar.

🔎

Arama (FTS5)

Arsiv indeksinde hizli arama icin SQLite FTS5 (Full-Text Search) sanal tablosu kullanilir. Dosya adi, yol ve uzanti uzerinde anlik arama saglar.
